## Aims
The aim of this group assignment is to design and implement a virtual reality (VR) application using Unity with the XR Interaction Toolkit. The VR environment is to contain four rooms;
- The first three rooms contain a puzzle that the user must solve
  - They can only progress to the next room after solving the puzzle in the current room.
- The fourth and final room is a victory / trophy room.
  - It celebrates the user reaching the end of the challenge with a celebratory theme
  -It contains objects that the user can interact with as well as displaying credits for the group
